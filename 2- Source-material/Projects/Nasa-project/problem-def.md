# User Pain Problem: Space Habitat Design Tools

## 1. Time-Consuming Design Process
- Creating a **concept for a space habitat** requires evaluating **structural forms, layouts, volumes, and systems integration**.  
- Each iteration can take **days or weeks**, slowing down early exploration.  
- Example: A designer testing both **inflatable** and **metallic habitats** must redraw layouts and redo volume calculations in separate software tools.

- Tools like **CATIA**, **SolidWorks**, or **ANSYS** are powerful but:
